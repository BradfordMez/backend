# BackEndAfrMark

Endpoints :
```
[POST] /api/users/register
[POST] /api/users/login

[GET] /api/items/
[POST] /api/items/

[GET] /api/markets/
[GET] /api/markets/:market_id
[POST] /api/markets/
```
------------------------------------------------------------------------------------

```
[POST] (/api/users/register) :
// body

  "username": "foo",
  "name": "foo Name",
  "password": "1234",
}

}
```

Get Items 
```
[GET] (/api/items/) :
// response
{
        "item_id": 1,
        "market_id": 1,
        "item_description": "Description would be here",
        "item_name": "Granny Smith",
        "item_price": 1.2
}
```
Add Items (must be logged in)
```
[POST] (/api/items/) :
// body
{
        "market_id": 1,
        "item_description": "Description would be here",
        "item_name": "Granny Smith",
        "item_price": 1.2
    }
// response
{
    "item_id": 11,
    "item_name": "Granny Smith",
    "item_description": "Description would be here",
    "item_price": 1.2,
    "market_id": 1
}
```
